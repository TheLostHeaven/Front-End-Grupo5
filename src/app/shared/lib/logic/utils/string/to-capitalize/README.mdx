# toCapitalize

La función `toCapitalize` convierte una cadena de texto, capitalizando la primera letra de cada palabra y convirtiendo el resto de las letras a minúsculas. Esta función es útil para formatear títulos, nombres propios y otros textos que requieren capitalización.

## Uso

La función `toCapitalize` toma una cadena de texto como entrada y devuelve la cadena capitalizada.

## Ejemplo de Uso

Puedes usar `toCapitalize` para formatear texto en un formato de título:

```typescript
import { toCapitalize } from '@shared/lib/logic/toCapitalize/toCapitalize';

const title = toCapitalize('hello world');
console.log(title); // Output: "Hello World"
```
