## Componente power-fix-input

Componente para la implementación de inputs

### Importación del módulo

```javascript
import { PowerFixInputModule } from '@shared/lib/ui';
@NgModule({
  ...
  imports: [
    ...
    PowerFixInputModule,
    FormsModule,
    ReactiveFormsModule,
  ],
})
```

```javascript
import { FormBuilder, FormGroup, Validators } from '@angular/forms';

formGroup?: FormGroup;

constructor(private formBuilder: FormBuilder) {
  this.formGroup = this.formBuilder.group({
    testing_input: ['', [Validators.required]],
  });
}
```

```html
<form [formGroup]="formGroup">
  <power-fix-input
    formControlName="input_name"
    [type]="text"
    [placeholder]="'placeholder'"
    [messageInformative]="'messageInformative'"
    [messageError]="'messageError'"
    [textInformative]="'textInformative'"
    [iconName]="'iconName'"
  >
  </power-fix-input>
</form>
```

#### Iconos de PrimeNG

El componente `power-fix-input` admite el uso de iconos provenientes tanto de SherpaFont como de PrimeNG. Para utilizar los iconos de PrimeNG, puedes consultar la [biblioteca de iconos de PrimeNG](https://primeicons.org/), donde encontrarás una amplia gama de opciones disponibles. Simplemente puedes utilizar el nombre del icono de PrimeNG como valor para la propiedad `icon` del componente.
